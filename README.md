# OpenCV Project

## Prerequisites
- **Python**: 3.7.9
- **pip** (comes with Python)
- **Virtual Environment** (recommended)
- Install dependencies from `requirements.txt`:
  ```bash
  pip install -r requirements.txt
  ```
